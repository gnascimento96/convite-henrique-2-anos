# Convite Henrique — 2 anos

Convite interativo para o aniversário de 2 anos do Henrique.

O site publicado usa o arquivo `index.html`.

As imagens ficam em `assets/images` e são carregadas sob demanda para reduzir o uso de memória em celulares antigos. A figurinha inicial é priorizada; as fotos do álbum, o selo e o QR Code são carregados quando se aproximam da área visível.

O arquivo `404.html` redireciona endereços incompletos ou copiados com caracteres extras para a página inicial do convite.
